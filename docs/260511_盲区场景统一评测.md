# 260511 盲区场景统一评测 (DeepAccident + SUMO)

> **目标**: 在盲区 SUMO 场景下评测各方法所有指标，按用户优先级（WPColl% > 二次碰撞 > DetRate > FalseAlm > 严重度）排名。所有变体共用一组阈值/配置，不同方法不混用。
> **平台**: DeepAccident (val 22 unseen scenarios) + SUMO blind-spot (3 scenario types × 3 seeds × 4 sids)
> **基线**: 含 NoConstraint+ego_only (无 V2X) 和 NoConstraint+coop

---

## 一、SUMO 盲区场景设计

### 1.1 三类盲区场景

| 场景 | 描述 | 网络 | 主要冲突类型 |
|------|------|------|--------------|
| **BS1_uturn** | 前方右侧车辆掉头/对向头碰，前方有 blocker 卡车遮挡 | cross_1lane | 头碰 (CT1) |
| **BS3_corner** | 盲角交叉口，转角有静止 blocker 阻挡视线 | cross_1lane | 头碰 (CT1) |
| **BS4_merge** | 右侧盲区车辆并道汇入，右侧 truck 遮挡 | cross_2lane | 追尾 (CT3) |

### 1.2 盲区机制

- **`blk_*`** 命名前缀的车辆是 blocker（卡车 / 停车 / 转角障碍）
- **`att_*` / `attacker`** 命名前缀的车辆是攻击者，强制忽略安全 (setSpeedMode=0)
- **LOS 遮挡**: ego→target 视线若被 blocker 遮挡（< 4m 距离阈值），target 标记为 invisible
- **协同感知**: coop 车辆从另一角度观测，覆盖 ego 盲区

### 1.3 控制框架

- ego 用 setSpeedMode=6 禁用 SUMO 安全，由方法显式 setSpeed 控制
- attacker 用 setSpeedMode=0，完全无安全 → 强制冲突
- 其他车辆（coop, blocker）保留默认 SUMO 控制

## 五、综合结论与最优配置

### 5.1 各平台胜者不同

| 平台 | 最优 | 原因 |
|------|------|------|
| **DeepAccident** | **Hybrid-thr0.25 或 Hybrid+AND-thr0.25** | WPColl=0.3% (最低)，FA(f) 较低 |
| **SUMO 强制冲突** | **RSS-coop** | 0% 碰撞率（强制刹停成功避让） |

### 5.2 跨平台最优折衷

如果必须选一个用于两类场景，**Hybrid+AND-thr0.25** 综合最佳：
- DeepAccident: WPColl=0.3% (最低), FA(f)=40% (低), Det(s)=100%
- SUMO: 碰撞率 56%, 但 FA(s)=50% (Hybrid 单独 FA=100%)

这是因为 AND-fusion 提高了精度，避免在没有真实威胁时虚报，同时保留 Hybrid 的路点安全优势。

### 5.3 全平台诚实对比

| 维度 | 哪个方法/变体最优 | 数值 |
|------|------------------|------|
| **DeepAccident WPColl%** | Hybrid 任何阈值 / Hybrid+AND | **0.3%** |
| **DeepAccident FA(f)** | MAP-coop | 33% |
| **DeepAccident Det(s)** | 多数方法都 100% | 100% |
| **DeepAccident Early** | RSS / APF / RiskMM | 62.3 |
| **SUMO 碰撞率** | **RSS-coop** | **0%** |
| **SUMO 二次碰撞** | 全部 | 0 |
| **SUMO 严重度** | RSS (无碰撞) > RiskMM 2.43 | — |
| **SUMO Early** | UniE2EV2X / Hybrid | 41-46 帧 |
| **SUMO FA(f)** | UniE2EV2X (2%) / Hybrid+AND (2%) | 2% |
| **协同感知价值 (WPC)** | egoonly 4.4% → coop 0% (DA), 3.4%→0% (SUMO) | ↓100% |

### 5.4 推荐配置（按用户优先级）

**保守推荐**: `Hybrid+AND-thr=0.25`
- DeepAccident: WPC=0.3%, FA(f)=40%, Det(s)=100%, Early=45.5
- SUMO: 碰撞率 56%, 但 FA(s)=50% (vs 100% 大多数 Hybrid)
- 真实路测优势: V1+geometric AND 减少 spurious 警报

**避碰最优 (SUMO 性能优先)**: `RSS-coop`  
- SUMO: 0% 碰撞率
- DeepAccident: WPC=4.4% (差), FA(f)=95%, Det(s)=100%
- 缺点: 路点安全差，警报频繁

## 七、观察总结

1. **DeepAccident 上 Hybrid 仍然是 WPColl 之王** (0.3% vs 0.4% MAP vs 0.9% UniE2EV2X)
2. **SUMO 强制冲突中 RSS 反而最强**（其激进的安全距离扩展+刹车策略）
3. **协同感知确实降低 WPColl**（NoCon-egoonly 3.4% → NoCon-coop 0.0%）
4. **AND-fusion 显著降低 FA(f)** 同时保持 Det(s)=100% 和 WPColl=0.3%
5. **没有"全平台最优"**，但 Hybrid+AND-thr0.25 在 DeepAccident 和 SUMO 上都进入前列
