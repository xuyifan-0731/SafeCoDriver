# 实验流程完整性审计

## 审计目标

确认实验中**没有虚假/合成/估计数据**，所有数据来源可追溯。

## 逐项审计

### 1. DAIR-V2X 离线实验

| 数据项 | 来源 | 是否真实 | 说明 |
|--------|------|---------|------|
| 目标 3D 位置 (x, y) | cooperative/label_world/*.json | ✅ 真实 | DAIR-V2X 官方世界坐标 3D 标注 |
| 目标尺寸 (l, w, h) | 同上 | ✅ 真实 | 3D 框标注 |
| 目标类型 | 同上 | ✅ 真实 | car/van/truck/bus/pedestrian/cyclist |
| 目标旋转角 | 同上 | ✅ 真实 | rotation 字段 |
| **ego 位置** | vehicle-side/calib/novatel_to_world/*.json | ✅ 真实 | GPS+IMU 融合的世界坐标 |
| **ego 速度** | 连续帧 novatel_to_world 位置差 / Δt=0.1s | ✅ 真实计算 | 98.9% 帧有连续帧 |
| **agent 速度** | 连续帧同一目标位置差 / Δt=0.1s（按最近邻匹配） | ✅ 真实计算 | 95% agents 匹配成功 |
| 无法匹配的 agent 速度 | 设为 0 m/s | ⚠️ 保守估计 | 5% agents，不假装有速度 |
| **车道信息** | **无** | ✅ 不使用 | DAIR-V2X 无车道标注，实验不使用车道约束 |
| 协同 vs 非协同分离 | vehicle-side/label/ vs cooperative/label_world/ | ✅ 真实分离 | DAIR-V2X 提供的独立标注 |

**结论**: DAIR-V2X 实验数据全部真实，无合成/估计内容（除 5% 无法匹配的 agent 速度设为 0）。

### 2. CARLA 闭环实验

| 数据项 | 来源 | 是否真实 | 说明 |
|--------|------|---------|------|
| 所有 actor 位置/速度 | CARLA API get_transform()/get_velocity() | ✅ 仿真真值 | Ground truth from simulator |
| 碰撞检测 | CARLA sensor.other.collision | ✅ 仿真真值 | 物理引擎碰撞检测 |
| 行人行为 | CARLA controller.ai.walker | ✅ 仿真生成 | AI 行人控制器 |
| 交通行为 | CARLA TrafficManager (aggressive) | ✅ 仿真生成 | 配置：近距跟车、超速、闯灯 |
| ego 控制 | autopilot + 安全约束覆盖 | ✅ 真实执行 | 约束生效时覆盖 autopilot 控制 |
| 车道信息 | **未使用** | ✅ | 闭环中不使用车道约束 |

**结论**: CARLA 实验数据全部来自仿真器真值，无合成/估计内容。

### 3. 已清除的问题

| 原问题 | 状态 | 处理方式 |
|--------|------|---------|
| DAIR-V2X 速度是假的 (v1 loader) | ✅ 已修复 | v2 loader 用真实连续帧计算 |
| 协同对比随机丢弃 agent | ✅ 已修复 | 用 vehicle-side 真实标注 |
| DAIR-V2X 合成车道 (v1 loader) | ✅ 已移除 | v2 loader 不生成车道，不使用车道约束 |
| 合成场景 (scenarios.py) | ✅ 不再使用 | 所有实验用真实数据或仿真器 |

### 4. 仍需注意的事项

| 事项 | 说明 | 影响 |
|------|------|------|
| DAIR-V2X 无车道标注 | 可行域仅由动力学可达集决定，不受道路约束 | 可行域比有车道时更大 |
| CARLA Town10HD 较开阔 | 不是最密集的交通场景 | 碰撞率可能低于真实城市 |
| CARLA autopilot 避碰能力强 | 基线方法"不碰撞"部分归功于 autopilot 而非安全约束 | 闭环碰撞率对比需谨慎解读 |
| 5% agents 速度为 0 | 可能低估这些 agent 的风险 | 对约束率有微小影响 |

## 结论

当前实验流程中**不存在虚假/合成数据**。所有数据来源可追溯到 DAIR-V2X 官方标注或 CARLA 仿真器真值。
