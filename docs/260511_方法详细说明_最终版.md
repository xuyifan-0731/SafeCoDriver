## 一、方法概述

### 1.1 设计哲学

**核心原则: 检测与修正解耦, 同时输出"减速强度"信号**。

```
                ┌──────────────────────────────┐
  perception ──→│ HybridSafetyConstraint      │
                │                              │
                │ ┌─ V1 collision detector ──┐│──→ collision_prob
                │ │                          ││
                │ ├─ Geometric threat finder ┤│──→ n_geometric_threats
                │ │                          ││
                │ ├─ TTC computer            ┤│──→ min_ttc
                │ │                          ││
                │ ├─ Multi-agent repulsion   ┤│──→ modified waypoints
                │ │                          ││
                │ └─ Brake intensity decider ┘│──→ target_speed_factor
                └──────────────────────────────┘
                         │     │
                  detection_flag, brake_factor, modified_wp
```

每帧产出三类信号:
1. **`collision_prob`** (来自 V1 网络): 决定是否报警 (DetRate / FA)
2. **`modified waypoints`** (来自几何修正): 决定路点安全性 (WPColl%)
3. **`target_speed_factor`** (来自 P0+P1 决策树): 决定闭环刹车强度 (CollRate)

### 1.2 三大创新点

| 创新 | 解决的问题 | 实现位置 |
|------|----------|---------|
| **可见性感知边距** (visibility-aware margin) | 协同感知中 ego-only 的 agent 位置不确定性 | `_get_safety_margin` |
| **多智能体排斥力** (multi-agent repulsion) | 多个威胁 agent 同时存在时的路点修正 | `_multi_agent_repulsion` |
| **检测-修正解耦** (decoupled det-mod) + **显式刹车** (P0+P1) | 路点修正 ≠ 闭环刹车，需独立信号 | `constrain_waypoints` 末端 |

### 3.2 主入口: `constrain_waypoints`

输入: `waypoints` (10×2 数组), `perception` (PerceptionResult)
输出: `(modified_waypoints, stats_dict)`

**完整流程**:

```python
# 文件: coop_safety/learned/hybrid_safety.py
# 行: 230-326

def constrain_waypoints(self, waypoints, perception):
    modified = waypoints.copy()
    ego = perception.ego
    ego_speed = max(ego.velocity, 1.0)
    n_geometric_threats = 0

    # —— 步骤 1: V2 路点风险 (可选, 大多场景不启用) ——
    wp_risks = self._get_wp_risks(waypoints, perception)

    # —— 步骤 2: 几何威胁检测 + 多智能体排斥 (始终运行) ——
    for t in range(len(waypoints)):
        threats = self._find_threats(modified[t], perception.agents,
                                     t, ego_speed)
        if threats:
            n_geometric_threats += 1
            modified[t] = self._multi_agent_repulsion(
                modified[t], threats,
                wp_risk=wp_risks[t] if t < len(wp_risks) else 0.0)

    # —— 步骤 3: 轨迹平滑 ——
    if self.smooth_weight > 0:
        smoothed = modified.copy()
        for t in range(1, len(modified) - 1):
            mid_x = (modified[t-1, 0] + modified[t+1, 0]) / 2
            mid_y = (modified[t-1, 1] + modified[t+1, 1]) / 2
            smoothed[t, 0] = modified[t, 0] * (1 - self.smooth_weight) + mid_x * self.smooth_weight
            smoothed[t, 1] = modified[t, 1] * (1 - self.smooth_weight) + mid_y * self.smooth_weight
        modified = smoothed

    # —— 步骤 4: 速度钳位 (相邻路点距离 ≤ v_max × dt) ——
    max_dist = self.v_max * WAYPOINT_DT
    for t in range(1, len(modified)):
        dx = modified[t, 0] - modified[t-1, 0]
        dy = modified[t, 1] - modified[t-1, 1]
        dist = math.sqrt(dx**2 + dy**2)
        if dist > max_dist:
            scale = max_dist / dist
            modified[t, 0] = modified[t-1, 0] + dx * scale
            modified[t, 1] = modified[t-1, 1] + dy * scale

    # —— 步骤 5: V1 网络检测 (与修正解耦) ——
    v1_prob = self._detect_with_v1(perception)
    is_dangerous = v1_prob > self.detection_threshold

    # —— 步骤 6 (P0+P1): TTC + 威胁计数 → 显式刹车强度 ——
    min_ttc = self._compute_min_ttc(perception)
    if min_ttc < 2.0 or (n_geometric_threats >= 2 and v1_prob > 0.5):
        target_speed_factor = 0.1   # EMERGENCY: 90% 减速
    elif min_ttc < 3.0 or n_geometric_threats >= 2 or v1_prob > 0.7:
        target_speed_factor = 0.3   # HARD_BRAKE: 70% 减速
    elif min_ttc < 5.0 and (n_geometric_threats >= 1 or v1_prob > self.detection_threshold):
        target_speed_factor = 0.5   # CONSERVATIVE: 50% 减速
    elif n_geometric_threats >= 1 or v1_prob > self.detection_threshold:
        target_speed_factor = 0.7   # CAUTIOUS: 30% 减速
    else:
        target_speed_factor = 1.0   # NORMAL

    return modified, {
        "n_collisions_detected": 1 if is_dangerous else 0,
        "modification_rate": n_geometric_threats / max(len(waypoints), 1),
        "collision_prob": v1_prob,
        "n_geometric_threats": n_geometric_threats,
        "target_speed_factor": target_speed_factor,   # ← P0+P1 关键输出
        "min_ttc": min_ttc,
        "mode": mode_str,
    }
```

### 3.3 辅助方法详解

#### 3.3.1 `_get_safety_margin` (行 91-109)

**视性感知 + 接近速度自适应安全边距**:

```python
def _get_safety_margin(self, agent, ego_speed: float) -> float:
    if agent.is_visible:
        margin = self.base_margin_visible          # 2.5m
    else:
        margin = self.base_margin_invisible        # 4.0m (盲区 +60%)

    s = agent.state
    dist = math.sqrt(s.x**2 + s.y**2)
    if dist > 0.01:
        rel_vx = s.vx - ego_speed
        rel_vy = s.vy
        approach_speed = -(s.x * rel_vx + s.y * rel_vy) / dist
        if approach_speed > 0:
            # 高速接近 → 边距增大 (max +30% at v=v_max)
            margin *= (1.0 + self.approach_speed_factor *
                      min(approach_speed / self.v_max, 1.0))

    margin += max(s.length, s.width) * 0.3   # 大型车额外缓冲
    return margin
```

公式:

$$
m(A_i) = m_{\text{base}}^{\text{vis}}(A_i) \cdot \left(1 + \beta \cdot \min\left(\frac{\max(v_{\text{app}}, 0)}{v_{\max}}, 1\right)\right) + 0.3 \cdot \max(\ell_i, w_i)
$$

其中 $m_{\text{base}}^{\text{vis}}=2.5$m (visible), $m_{\text{base}}^{\text{invis}}=4.0$m, $\beta=0.3$, $v_{\text{app}}$ 为 agent 相对 ego 的接近速度。

#### 3.3.2 `_find_threats` (行 111-124)

**对每个路点 wp[t] 在时刻 t 预测 agent 位置, 检查是否进入 wp 的安全边距**:

```python
def _find_threats(self, wp, agents, t_idx, ego_speed):
    dt = (t_idx + 1) * WAYPOINT_DT      # t=0 → dt=0.5s, t=9 → dt=5s
    threats = []
    for agent in agents:
        a = agent.state
        # 常速预测 agent 位置
        ax = a.x + a.vx * dt
        ay = a.y + a.vy * dt
        dist = math.sqrt((wp[0] - ax)**2 + (wp[1] - ay)**2)
        margin = self._get_safety_margin(agent, ego_speed)
        if dist < margin:
            threats.append((agent, ax, ay, dist, margin))
    return threats
```

注意: agent.state.vx/vy 已经是 **ego 帧下的相对速度** (在 `build_perception` 中转换), 所以 `a.x + a.vx*dt` 直接得到 t 时刻 agent 在 ego 帧 (随 ego 一起移动) 中的位置。

#### 3.3.3 `_multi_agent_repulsion` (行 126-155)

**多智能体排斥力合成路点偏移**:

```python
def _multi_agent_repulsion(self, wp, threats, wp_risk=0.0):
    if not threats:
        return wp.copy()

    total_fx, total_fy = 0.0, 0.0
    max_push = 0.0
    for agent, ax, ay, dist, margin in threats:
        dx = wp[0] - ax     # 从 agent 指向 wp 的方向 (排斥)
        dy = wp[1] - ay
        d = max(dist, 0.1)
        # 力大小: 越接近边距越强, 1/d 加权
        force_mag = (margin - dist + self.push_clearance) / d
        # 视性权重: 不可见 agent 排斥力 ×1.5
        vis_weight = 1.0 if agent.is_visible else 1.5
        # V2 风险权重 (默认 1.0)
        risk_weight = 1.0 + wp_risk
        weight = vis_weight * risk_weight * force_mag

        total_fx += (dx / d) * weight
        total_fy += (dy / d) * weight
        max_push = max(max_push, margin - dist + self.push_clearance)

    # 归一化方向 + 推开 max_push 距离 (确保至少 1m)
    force_norm = math.sqrt(total_fx**2 + total_fy**2)
    if force_norm > 0.01:
        push_dist = max(max_push, 1.0)
        new_wp = wp.copy()
        new_wp[0] += (total_fx / force_norm) * push_dist
        new_wp[1] += (total_fy / force_norm) * push_dist
        return new_wp
    return wp.copy()
```

公式:

$$
\mathbf{F}_i = \frac{(\mathbf{wp} - \hat{\mathbf{p}}_i)}{\|\mathbf{wp} - \hat{\mathbf{p}}_i\|} \cdot \frac{m_i - d_i + c}{d_i} \cdot w_{\text{vis}}^i \cdot (1+r_{\text{wp}})
$$

$$
\Delta \mathbf{wp} = \frac{\sum_i \mathbf{F}_i}{\|\sum_i \mathbf{F}_i\|} \cdot \max\left(\max_i (m_i - d_i + c), 1.0\right)
$$

#### 3.3.4 `_detect_with_v1` (行 157-187)

**V1 网络推理: 输出场景级碰撞概率**:

```python
def _detect_with_v1(self, perception):
    if self.detector is None:
        return 0.0
    try:
        agents_feat = np.zeros((1, 30, 10), dtype=np.float32)
        mask = np.zeros((1, 30), dtype=bool)
        for i, a in enumerate(perception.agents[:30]):
            s = a.state
            agents_feat[0, i] = [
                s.x, s.y, s.vx, s.vy, s.heading,
                s.length, s.width, s.velocity,
                1.0 if a.is_visible else 0.0, 0
            ]
            mask[0, i] = True
        with torch.no_grad():
            cp, _ = self.detector(
                torch.FloatTensor(agents_feat),
                torch.BoolTensor(mask))
        return cp.item()
    except Exception as e:
        logger.warning(f"V1 inference failed: {e}")
        return 0.0
```

**特征向量** (10 维 per agent): `[x, y, vx, vy, heading, length, width, |v|, is_visible, type_pad]`

#### 3.3.5 `_compute_min_ttc` (行 209-228) — P1 新增

**最小 TTC 计算**:

```python
def _compute_min_ttc(self, perception):
    min_ttc = 999.0
    for a in perception.agents:
        s = a.state
        dist = math.sqrt(s.x**2 + s.y**2)
        if dist < 0.5:
            return 0.0   # 已经重叠
        if dist > 0.1:
            # 接近速度: 相对速度沿 ego→agent 方向投影
            approach = -(s.x * s.vx + s.y * s.vy) / dist
            if approach > 0.1:    # agent 正在靠近
                ttc = dist / approach
                if ttc < min_ttc:
                    min_ttc = ttc
    return min_ttc
```

公式:

$$
v_{\text{app}}^i = -\frac{\mathbf{p}_i \cdot \mathbf{v}_i}{\|\mathbf{p}_i\|}, \quad
\text{TTC}_i = \begin{cases} \frac{\|\mathbf{p}_i\|}{v_{\text{app}}^i} & v_{\text{app}}^i > 0.1 \\ \infty & \text{otherwise} \end{cases}
$$

$$
\text{TTC}_{\min} = \min_i \text{TTC}_i
$$

## 四、V1 网络详解 (`collision_network.py`)

### 4.1 架构 (122 行, 46,659 参数)

```python
# 文件: coop_safety/learned/collision_network.py

class AgentEncoderV2(nn.Module):  # 输入: (B, N, 10) → (B, N, 64)
    def __init__(self, in_dim=10, hidden=64):
        self.mlp = nn.Sequential(
            nn.Linear(in_dim, hidden), nn.ReLU(),
            nn.Linear(hidden, hidden), nn.ReLU())

class InteractionModule(nn.Module):  # 4-head self-attention
    def __init__(self, dim=64, n_heads=4):
        self.attn = nn.MultiheadAttention(dim, n_heads, batch_first=True)
        self.norm = nn.LayerNorm(dim)

    def forward(self, x, mask):
        attn_out, _ = self.attn(x, x, x, key_padding_mask=~mask)
        return self.norm(x + attn_out)

class AttentionPool(nn.Module):  # (B, N, 64) → (B, 128)
    def __init__(self, dim=64, out_dim=128):
        self.attn_weight = nn.Linear(dim, 1)
        self.proj = nn.Linear(dim, out_dim)

    def forward(self, x, mask):
        scores = self.attn_weight(x).squeeze(-1).masked_fill(~mask, -1e9)
        weights = F.softmax(scores, dim=-1)
        pooled = (x * weights.unsqueeze(-1)).sum(dim=1)
        return self.proj(pooled)

class CollisionPredictionNetwork(nn.Module):  # 主网络
    def __init__(self, agent_feat_dim=10, hidden=64, scene_dim=128):
        self.encoder = AgentEncoderV2(agent_feat_dim, hidden)
        self.interaction = InteractionModule(hidden, n_heads=4)
        self.pool = AttentionPool(hidden, scene_dim)
        self.collision_head = nn.Sequential(
            nn.Linear(scene_dim, 64), nn.ReLU(),
            nn.Linear(64, 1))      # Sigmoid 在 forward 中
        self.ttc_head = nn.Sequential(
            nn.Linear(scene_dim, 64), nn.ReLU(),
            nn.Linear(64, 1))      # ReLU 在 forward 中

    def forward(self, agents, mask, ego=None):
        x = self.encoder(agents, mask)
        x = self.interaction(x, mask)
        scene = self.pool(x, mask)
        coll_prob = torch.sigmoid(self.collision_head(scene))
        ttc = F.relu(self.ttc_head(scene))
        return coll_prob, ttc
```

```python
def is_los_blocked(ego_pos, target_pos, blockers_pos, radius=BLOCKER_RADIUS):
    """检查 ego→target 视线是否被任意 blocker 阻挡 (距视线 < radius)。"""
    ex, ey = ego_pos; tx, ty = target_pos
    seg_dx = tx - ex; seg_dy = ty - ey
    seg_len = math.sqrt(seg_dx**2 + seg_dy**2)
    if seg_len < 0.1: return False
    for bx, by in blockers_pos:
        bdx = bx - ex; bdy = by - ey
        # blocker 在 ego→target 线段上的投影参数 t∈[0,1]
        t = (bdx*seg_dx + bdy*seg_dy) / (seg_len*seg_len)
        if t < 0 or t > 1: continue
        # blocker 到线段的距离
        proj_x = ex + t*seg_dx; proj_y = ey + t*seg_dy
        dist_to_seg = math.sqrt((bx-proj_x)**2 + (by-proj_y)**2)
        if dist_to_seg < radius:
            return True
    return False
```

ID 前缀约定:
- `ego`: 主车
- `coop`: 协同感知车 (另一视角)
- `att_*` 或 `attacker`: 攻击者 (强制冲突, setSpeedMode=0)
- `blk_*`: 障碍物 (创建盲区, setSpeedMode 不变)

## 九、用于论文的分析说明

### 9.1 主要贡献

> 本文提出 SafeCoDriver — 一个独立的、可插拔的协同感知安全约束模块，由三个关键组件构成：
> (1) **可见性感知几何检测** ($m^{\text{vis}}=2.5$m, $m^{\text{invis}}=4.0$m)，体现协同感知中盲区 agent 的位置不确定性；
> (2) **多智能体排斥力路点修正** $\mathbf{F}_i = (\mathbf{p}_{wp}-\mathbf{p}_i)/d_i^2 \cdot w_i$，处理多威胁同时发生的复杂场景；
> (3) **检测-修正-刹车三信号解耦架构**，每帧同时输出独立的检测概率、修正路点、显式刹车强度因子，使闭环控制器能够同时获得所有所需信号。

### 9.2 实验设计

> 我们在两个互补的平台上评测：
> - **DeepAccident** (Wang et al. 2023): 离线、配对场景 (52 accident + 52 normal)，使用 GT 感知，评测路点级安全性 (WPColl%) 和检测精度。
> - **SUMO 盲区仿真** (本文设计): 闭环、强制冲突场景 (单/双车道，attacker setSpeedMode=0)，包含 BS1 头碰盲区、BS3 转角盲区、BS4 右侧并道盲区，评测真实碰撞率和闭环控制有效性。
> 两个平台共用同一 V1 网络 (按场景划分 train/val，无数据泄漏，AUC=0.81)。

### 9.3 关键结果

> **DeepAccident**: SafeCoDriver 路点碰撞率 (WPColl%) 0.3%，相对于 MAP (0.4%)、UniE2EV2X (0.9%)、RSS (4.4%) 实现 25-93% 改进，同时保持 100% 场景级检测率。
> **SUMO 盲区**: SafeCoDriver 真实碰撞率 0% (vs NoCon 56%, MAP 56%, UniE2EV2X 56%, RSS 0%, RiskMM 33%)。RSS 同样达到 0% 但路点级 WPColl=4.4% (无路点修正能力)。
> **SafeCoDriver 是首个在两个平台上同时获得最优指标的方法**。

### 9.4 消融分析 (核心)

| 组件 | DA WPC% | SUMO CollRate |
|------|---------|---------------|
| 仅 V1 (无几何修正) | 4.4% | 56% |
| 仅几何 (无 V1) | 0.3% | 56% |
| 几何 + V1 (无 P0) | 0.3% | 56% |
| 几何 + V1 + **P0** | 0.3% | **0%** |
| 几何 + V1 + **P0+P1** | 0.3% | **0%** |

> 消融显示几何修正主导 WPColl% 改进, P0 的显式 brake factor 信号主导 SUMO CollRate 改进。两者独立起作用，验证了三信号解耦架构的合理性。

### 9.5 局限性 (诚实)

1. V1 网络在 DeepAccident 训练 (CARLA 仿真)，对真实路测数据 (DAIR-V2X) 存在 domain gap，FA 较高，需要 fine-tune。
2. SUMO 强制冲突场景中，RSS 同样达到 0% CollRate；但 RSS 在 DA 上无路点修正能力。SafeCoDriver 的优势在于**同时**做好两件事。
3. SUMO 评测覆盖 3 类盲区场景 × 3 种子 × 4 sids = 36 runs，统计性可在更大样本上加强。
